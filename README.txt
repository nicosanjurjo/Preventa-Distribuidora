 Este sistema de gestión de pedidos fue desarrollado, a medida, para una distribuidora local de productos varios.
 
 Se puede probar ejecutando el servidor local con Django, previa activación del entorno virtual e instalación de sus dependencias especificadas en "requirements.txt".
Accediendo al directorio raíz, se podrán dar de alta clientes y realizar pedidos, que luego serán visualizados y gestionados por 
el administrador accediendo al directorio raíz seguido de "/gestion", por ejemplo: "localhost:8000" y "localhost:8000/gestion" respectivamente.
 
 La base de datos presente solo contiene Productos, Clientes y Vendedores ficticios para que se pueda probar la funcionalidad básica del sistema
realizando pedidos (como lo haría el preventista) y gestionándolos
(como lo haría el administrador).
